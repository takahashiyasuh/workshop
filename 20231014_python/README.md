## ワークショップ

2023-10-14 に実施したワークショップです。<br>
Pythonを使ってDBからデータを取得してEXCELにグラフを書かせてみます。

## 開発環境

- Python3系
- MySQL

パッケージ

- openpyxl

ODBCは MySQL からインストールでも mysqlclient を使うでもどちらでもOK。
